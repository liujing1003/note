# 内联函数

  编译器把整个函数的代码直接插在调用语句处，不会产生调用函数的语句。

  在函数定义前面加”inline"关键字，即可定义内联函数

          inline int Max(int a,int b)
           {
             if(a>b) return a;
             return b;
           }

# 函数重载
* 一个或多个函数，名字相同，然而参数个数或参数类型不相同，就叫做函数的重载

例如：

      int Max(double f1,double f2){}
      int Max(int n1,int n2){}
      int Max(int n1,int n2,int n3){}
