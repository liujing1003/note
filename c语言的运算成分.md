# 赋值运算符
> 两边类型不同：

    (1)若=两边类型不一致，赋值时要进行类型转换
    (2)不管=右边的操作数是什么类型，都转化为左边的类型

> 长数赋给短数：

    (1)截取长数的低n位送给短数

> 短数赋给长数

     (1)原来是什么数现在还是什么数
     (2)short为无符号数，高16位补0
     (3)short为有符号数，若short型最高位为0，则long型高16位补0；最高位为1，则long型高16位补1；

> 符号位的赋值处理：

　　　直接赋值，不管符号位还是数字位

# 算术运算符:
* １.整数运算，结果仍为整数

　　　　实数运算，结果为double型

> 自增自减

    (1)++i,--i

        在使用i之前,先将i的值加(减)1

    (2)i++,i--

        在使用i之后,再将i的值加(减)1

> 混合运算的优先级;

   !高于算术运算符高于关系运算符高于&&和||高于赋值运算符.

# 条件运算符;

   表达式1?表达式2:表达式3

   规则: 如果1为真,则以2的值作为表达式的值;否则以表达式3的值作为整个表达式的值.

# 强制类型转换;

> 形式;

    (类型名)(表达式)
    强制类型转换后,被转换的量的类型并没有发生变化,
